Demo 2: Motion Profiles

Demonstrates the actual MotionProfile.ts trapezoidal motion system used by Oscillators for smooth movement.

Motion Profile Phases

Acceleration Ramp up to max velocity (blue phase)
Cruise Constant velocity travel (green phase)
Deceleration Ramp down to zero velocity (red phase)

What You're Seeing

Position Angular displacement (θ) - where the Mobile is
Velocity Rate of change (dθ/dt) - speed of movement
Acceleration Rate of velocity change (d²θ/dt²) - force applied

Key Concepts

Smooth Motion Trapezoidal profiles provide smooth starts and stops
Real Implementation This uses the actual MotionProfile.ts subsystem
Motion Profile Controls
Total Distance 180°
Max Velocity 60°/s
Max Acceleration 60°/s²
