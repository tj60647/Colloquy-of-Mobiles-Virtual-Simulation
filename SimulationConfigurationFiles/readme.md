# Simulation Configuration Files

This directory contains JSON configuration files that define various settings and parameters for the simulation.

## Contents

- **config.json**: The default configuration file used for standard simulation runs.
- **config_240812.json**: A specific configuration file tailored for a particular scenario or test run (e.g., generated on August 24, 2012).

### How to Use

1. **Default Configuration**: To run the simulation with default settings, use `config.json`.
2. **Custom Configuration**: To run a simulation with custom settings, specify the desired configuration file when starting the application.

### Adding New Configurations

- To add a new configuration file, create a new `.json` file in this directory and follow the structure provided in `config.json`.
- Make sure to update this README with details about the new configuration.
