# Colloquy of Mobiles Virtual Simulation

This project is a virtual simulation of mobile entities that interact and communicate with each other. The goal is to model and analyze the behavior of these entities in a simulated environment.

## Directory Structure

- **docs/**: Contains all project documentation, including user manuals and design diagrams.
- **SimulationConfigurationFiles/**: Stores JSON configuration files used to control various aspects of the simulation.
- **src/**: Contains the source code for the application, including main code and tests.
  - **main/**: The core logic and functionality of the simulation.
  - **test/**: Unit tests, integration tests, and other test-related code.
