# Assets Directory

This directory contains shared assets used across the Colloquy of Mobiles Virtual Simulation project. These assets include fonts, images, stylesheets, and any other static resources that may be required by various demos or the main application.

## Directory Structure

The `assets/` directory is organized as follows:

```plaintext
assets/
├── fonts/
│   ├── CustomFont.ttf
├── images/
│   ├── logo.png
│   ├── background.jpg
└── styles/
    ├── main.css
```
